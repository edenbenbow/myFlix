Watchr documentation

This web app is designed to provide users with access to information about different movies, directors, and genres.

Users will be able to sign up, update their personal information, and compile a list of their favorite movies.

Request URL HTTP method Query parameters Request body data format Response data format
Return a list of all movies /movies GET None None JSON object holding data about all movies. Example:
{ title : "Head of State", director: "Chris Rock", genre: "Comedy", description: 'Chris Rock plays Mays Gilliam, a Washington, D.C. alderman who becomes the Democratic Party\'s longshot presidential candidate.' }
Return data (description, genre, director, image URL) about a single movie by title /movies/[title] GET None None JSON object showing movie data including genre, name, and description. (See above.)
Return data about a genre (description) by name/title /genres/[name] GET None None JSON object holding data about the genre, showing name and description. Example:
{ name: "Mystery", description: "In this genre, detectives or other professionals solve crimes." }
Return data about a director (bio, birth year) by name /directors/[name] GET None None JSON object holding data about the director
Allow new users to register /users POST None JSON object that contains new user data.
Example:
{ "username": "janeanderson", "password": "12345", "email": "janeanderson@gmail.com", "dob": "1/1/2001" } 		JSON object showing new user data, including username, password, email, and date of birth
Allow users to update their user info /users/[username] PUT None JSON object. Example:
{ "username": "johnsmith", "password": "12345", "email": "johnsmith@gmail.com", "dob": "1/1/2001" }		 A text message confirming the user info was updated
Allow users to add a movie to their list of favorites /users/[username]/favorites/[title] POST None None JSON object showing submitted movie data
Allow users to remove a movie from their list of favorites /movies/[username]/favorites/[title] DELETE None None A text message confirming the movie was removed
Allow existing users to deregister /users/[username] DELETE None None A text message confirming the account was deleted
